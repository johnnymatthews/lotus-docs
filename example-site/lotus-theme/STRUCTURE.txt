## Key Features

1. **Responsive Design** - Mobile-first, works on all devices
2. **Dark Mode** - Auto-detecting with manual toggle
3. **Code Highlighting** - Syntax highlighting with tab support
4. **Documentation-focused** - Sidebar navigation, TOC, breadcrumbs
5. **Multilingual Ready** - i18n support built-in
6. **Search** - Built-in search functionality
7. **Customizable** - Easy to override templates and styles

## Template Inheritance

│   └── page-navigation.html (partial)
│
└── list.html (section listings)

    ├── sidebar.html (partial)
    └── breadcrumb.html (partial)

## Menus

The theme uses two main menu systems:

1. **main** - Top navigation bar
2. **docs** - Sidebar documentation menu (hierarchical)

Configure in config.toml under [[menus.main]] and [[menus.docs]]
